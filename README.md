# Inova Monitoring

This project provides a set of Python tools designed to monitor and display the current status of Inova Apps instances.

## Features

- **PostgreSQL Integration**: Query and analyze data directly from PostgreSQL databases.
- **Prometheus Metrics**: Fetch and process data from Prometheus endpoints.
- **Web Interface**: A simple, unified interface to overview the status of your instances.
- **SSO Authentication**: Secure access to the monitoring interface utilizing Single Sign-On (SSO) capabilities.
